# HamzaTech Aura X1 — Premium Single-Product eCommerce Platform

Welcome to the official repository of **HamzaTech**, a luxury, futuristic consumer technology brand. This project is a premium, production-ready, single-product eCommerce experience for our flagship product: the **HamzaTech Aura X1 Premium Wireless ANC Headphones**.

Designed to match the aesthetic standards of elite digital design agencies, the platform combines elegant 3D interactions, fluid micro-animations, and a highly optimized modern web stack.

---

## 💎 Brand Identity & Vision
**HamzaTech** sits at the intersection of high-fidelity audio engineering, futuristic product design, and minimalist luxury. Inspired by design languages like Apple, Nothing, Sony, Dyson, Stripe, Linear, and Framer, the brand prioritizes:
- **Exclusivity & Luxury**: Every interaction feels refined, deliberate, and high-end.
- **Futurism**: Dark, immersive UI paired with high-tech elements and holographic, glowing visuals.
- **Premium Simplicity**: Stripping away the noise to showcase the product with maximum visual impact.

The primary vision for this web platform is to create an immersive digital showroom for the **Aura X1**, providing a tactile, interactive purchase journey that elevates consumer desire.

---

## 🎧 Product Details: HamzaTech Aura X1
The **HamzaTech Aura X1** is a masterpiece of audio engineering and aesthetic design:
- **Hybrid Active Noise Cancellation (ANC)**: Industry-leading silence powered by a multi-microphone array.
- **Hi-Res Audio & Spatial Sound**: Custom-tuned 40mm dynamic drivers with dynamic head tracking.
- **Luxury Materials**: Lightweight aluminum, premium memory foam ear cushions, and tactile glass control dials.
- **Aura Ambient Lighting**: Subtle customizable orange glow rings integrated on the ear cups.
- **Battery Life**: Up to 40 hours of playtime with ANC enabled, plus ultra-fast USB-C charging.

## 🎨 Design Philosophy & Theme
Our design system is built around a **Premium Minimalist** aesthetic utilizing a stark dark and light contrast with vibrant orange accents.

- **Theme**: Dark Mode (Default) & Light Mode.
- **Color Palette**:
  - **Primary**: `#111111` (Deep Charcoal)
  - **Accent**: `#F97316` (Vibrant Orange / Amber Glow)
  - **Background Light**: `#FAFAFA`
  - **Background Dark**: `#0B0B0B`
  - **Card Light**: `#FFFFFF`
  - **Card Dark**: `#171717`
  - **Text Light/Dark**: `#111111` / `#FFFFFF`
  - **Secondary Text**: `#6B7280`
  - **Border**: `#E5E7EB`
- **Typography**:
  - **Headings**: `Manrope` (Clean, geometric sans-serif)
  - **Body**: `Inter` (Optimized for readability)

## ✨ Key Features & Interactive Aesthetics
This website implements premium, agency-grade 3D and interaction design rules:
- **Floating 3D Headphone Hero**: A central interactive 3D model of the Aura X1 that reacts to mouse movements.
- **Glassmorphism Cards**: Layered interface elements with soft blur, reflection effects, and subtle borders.
- **Orange Ambient Glow**: Radial gradient backdrops simulating a futuristic volumetric aura.
- **Mouse Parallax & Hover Lift**: Floating UI components that shift dynamically based on cursor coordinates.
- **Premium CTA Animations**: Fluid state changes, smooth page transitions, and elegant micro-interactions powered by Framer Motion.
- **Responsive Architecture**: Impeccable layouts optimized from ultra-wide desktops to mobile screens.

## 🛠️ Tech Stack
The platform uses a modern, performant, and type-safe development environment:
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router, React Server Components)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (Radix Primitives)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend / DB**: [Supabase](https://supabase.com/)
- **Authentication**: [Supabase Auth](https://supabase.com/docs/guides/auth)
- **Storage**: [Supabase Storage](https://supabase.com/docs/guides/storage) (Product assets & assets)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Form Validation**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Notifications**: [Sonner](https://sonner.emilkowal.ski/)
- **Deployment**: [Vercel](https://vercel.com/)
